# TSH and Free T4 GWAS and meta-analysis in AOU, PMBB, BioVU, BioMe, and ATLAS
- Ancestry-specific GWAS (EUR and AFR) with thyroid stimulating hormone (TSH) were conducted in five biobanks (AOU, UCLA ATLAS, Mount Sinai BioMe, Vanderbilt BioVU, and PMBB)
- Ancestry-specific meta-analysis were conducted
- The goal of this project is to highlight ancestry-specific genetic differences in TSH
- FreeT4 GWAS was also done
